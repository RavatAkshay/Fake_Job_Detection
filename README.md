# Fake_Job_Detection


## 📌 Project Overview

The Fake Job Detection System is a Machine Learning project designed to identify whether a job posting is *Fraudulent* or *Legitimate*. The project aims to help job seekers avoid fake job advertisements by analyzing job posting information and making intelligent predictions.

---

## 🎯 Objective

- Detect fake job postings using Machine Learning.
- Build an end-to-end prediction system.
- Compare multiple ML algorithms.
- Improve model performance using Hyperparameter Tuning.
- Understand which features influence predictions the most.

---

## 📂 Dataset

The dataset contains job posting information such as:

- Job Title
- Company Profile
- Job Description
- Requirements
- Benefits
- Employment Type
- Experience Required
- Education Required
- Industry
- Function
- Fraudulent (Target Variable)

Target:

- *0 → Legitimate Job*
- *1 → Fraudulent Job*
